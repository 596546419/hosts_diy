# hosts_diy
- 一个简单的host合并去重项目,去重后分成 domain host regex modify 四类保存
<!-- Badges -->
- 代办事项：
	1. 制作ui
	2. 尝试删除废弃的hosts
	3. 制作github action 自动化
- 使用方法：
	1. 将链接填入links.txt，每行一个，相同名字或没有名字的必须在链接后写上#名字
	2. 依次执行1.py 2.py 3.py
